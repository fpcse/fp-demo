# gtmtrackingtest.github.io
## Structure
Currently, there are 3 distinct html files following the below format. 
### Google Hosted Maps
### FP (MapLibre) Hosted Maps
[MapLibre Documentation](https://maplibre.org/maplibre-gl-js/docs/)
Goal of the MapLibre set-up is to show that the maps are fully functional and effectively no different from other map providers. The requirement to rebuild maps in the new format may be a detractor but provides added data security.
### Google and FP Hosted Videos
Defaulted to random video but can be changed at any time. Also shows the FP player is capable of doing everything a youtube hosted video can do.
